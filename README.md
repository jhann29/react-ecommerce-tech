# TechStore — E-commerce React

Prueba técnica React (#11): catálogo, detalle, carrito y checkout.

## Instalación

```bash
cd react-ecommerce
npm install
npm start
```

Abre [http://localhost:3000](http://localhost:3000)

## Rutas

| Ruta | Vista |
|------|-------|
| `/` | Catálogo |
| `/item/:id` | Detalle |
| `/cart` | Resumen (Brief) |
| `/checkout` | Checkout |
